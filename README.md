# Go Scales

Prints out the Notes for a specific scale.

## Usage

    $ scale C major
    C major scale: C D E F G A B C

    $ scale G minor
    G minor scale: G A A# C D D# F G

    $ scale F mixolydian
    F mixolydian scale: F G A A# C D D# F

## Installation

    go get -u github.com/mhutter/scale
